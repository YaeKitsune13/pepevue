# Логика сервиса по оформелинию заказов

## Файлы

1.  `cartservice.go`
2.  `orderservice.go`

## Логика

### cartservice.go

1.  AddToCart Добавление новых товаров
2.  UpdateCountCart Обновление количества товаров
3.  RemoveInCart Удаление товаров из корзины

### orderservice.go

> Функция всего 1 поочерёдное выполнение

1.  Взять все поля из таблицы Cart
2.  Создать новый Order(Заказ)
3.  Заполнить заказ товарами OrderItem(Товары)
4.  Закончить создание Order(Заказа)
5.  Очистка таблицы Cart

> Обязанность делать всё востанавливающимся через базу данных тоесть в случае ошибки откатить изменения через транзакций
